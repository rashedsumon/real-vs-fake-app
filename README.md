# Real vs Fake Faces Classifier (StyleGAN3)

An interactive Streamlit web application that detects whether an uploaded human face photo is a real person or an AI-generated DeepFake image generated by StyleGAN3.

## How it Works
1. **Automated Pipeline**: Upon initial deployment/startup, the app leverages `kagglehub` to securely download the **troykueh/real-vs-fake-faces-stylegan3** dataset.
2. **Transfer Learning**: It configures a PyTorch **MobileNetV3** model architecture to learn structural details.
3. **Streamlit UI**: Allows users to drop any face image into the browser and receive an instant categorization label alongside confidence readouts.

## Setup & Deployment Instructions
To run this project locally or via Streamlit Cloud:

1. Push this entire repository structure to your GitHub account.
2. Log into [Streamlit Cloud](https://share.streamlit.io/).
3. Connect your repository and select `streamlit_app.py` as your main file path.
4. Deploy! Streamlit will automatically install dependencies from `requirements.txt` and manage your workflow.
